# 外盘品种走势展示与预测
基于 Python 的外盘期货（黄金/原油等）价格走势可视化与短期预测项目

## 📖 项目简介
本项目是程序设计实践作业，目标是对外盘期货品种（如伦敦金、布伦特原油）进行数据获取、走势展示与价格预测，主要功能包括：
- 从聚宽/同花顺/akshare/雅虎财经等数据源获取外盘品种历史与实时数据
- 展示主力合约核心信息：最新价、增减仓、合约乘数、期限结构
- 绘制价格走势、期限结构等可视化图表
- 基于时间序列模型（如 ARIMA/移动平均）进行短期价格预测
